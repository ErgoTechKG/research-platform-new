# 科研管理平台课程模块 UI/UX 工作流设计文档

## 一、项目概述

### 平台定位
- **核心目标**: 建立本科科研实验班的数字化课程管理系统
- **默认语言**: 中文

### 用户角色
1. **学生** - 选课、学习、提交作业、参与项目
2. **教授** - 发布课程、评估学生、指导项目
3. **秘书** - 系统管理、数据统计、流程协调
4. **领导** - 决策支持、成果查看、政策制定
