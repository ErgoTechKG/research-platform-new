# 教授仪表盘

## 用户故事
作为教授，我需要专属的教学管理中心仪表盘，以便查看今日日程、待处理事项和学生管理概览。

## 功能描述
为教授提供教学管理中心界面，集中展示日程安排、待处理任务和学生管理信息。

## 详细需求
- 今日日程安排展示
- 待处理事项统计和提醒
- 学生管理概览信息
- 指导学生进度跟踪
- 最近提交状态显示
- 批量操作功能入口
- 报告导出功能
- 快速通知发送功能

## 验收标准
- [ ] 日程安排准确显示
- [ ] 待处理事项统计正确
- [ ] 学生信息实时更新
- [ ] 进度跟踪准确有效
- [ ] 批量操作功能完整
