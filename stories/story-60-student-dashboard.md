# 学生仪表盘

## 用户故事
作为学生，我需要专属的学习中心仪表盘，以便快速查看今日待办、课程进度、最新通知和成就统计。

## 功能描述
为学生提供个性化的学习中心界面，集中展示学习任务、进度、通知和成就信息。

## 详细需求
- 今日待办事项展示
- 课程进度可视化显示
- 最新通知和公告展示
- 快速入口功能按钮
- 成就统计和评级展示
- 个人设置和退出功能
- 截止时间提醒
- 学习数据统计

## 验收标准
- [ ] 待办事项实时更新
- [ ] 进度显示准确直观
- [ ] 通知展示及时有效
- [ ] 快速入口功能完整
- [ ] 成就统计准确展示
