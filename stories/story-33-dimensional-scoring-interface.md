# 分维度评分界面

## 用户故事
作为专家，我需要分维度评分界面，以便对思想品德、科技创新等分项评分。

## 功能描述
思想品德、科技创新等分项评分，提供专业的多维度评价功能。

## 详细需求
- 科技创新维度评审工作台
- 竞赛获奖自动计分
- 科研项目评分
- 创新能力专家评分
- 评分进度跟踪
- 评分指南查看
- 学生切换功能
- 评分确认和暂存
- 专家评语填写

- [ ] 评语功能正常
